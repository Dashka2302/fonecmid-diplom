Инструкция по начальной настройке
Для корректной работы новой функциональности необходимо зайти в базу от имени администратора и произвести следующие действия:
1.	Каждому сотруднику, который будет работать в 1С создать нового пользователя в Настройках пользователей и прав. В зависимости от выполняемого функционала пользователям назначить соответствующие права доступа (например, для специалистов – специалист, для менеджера – менеджер и т.д.).
2.	В справочник Сотрудники ввести всех сотрудников:
	Рядовым сотрудником необходимо выбрать Подразделение «Специалист», всем остальным – «Администрация»
	Должность выбрать в соответствии с занимаемой должностью
	Выбрать созданного для этого специалиста Пользователя информационной базой, если у него планируется доступ к базе
3.	В регистре сведений Условия оплаты сотрудникам для каждого сотрудника ввести соответствующий оклад и, в случае необходимости, процент от выполненных работ
4.	Заполнить график работы в обработке Заполнение графика:
	Выберите рабочий период, график на который у вас уже известен
	Введите порядковый номер дней недели, которые в этом периоде будут выходными (например, 6,7 – сб и вс – при пятидневной рабочей неделе, 7 – вс – при шестидневной)
5.	Заполнить документ График отпусков за рабочий год
6.	Назначить стандартные номенклатуры:
	В Номенклатура абонентская плата выбрать номенклатуру, которая будет назначена стандартной для соответствующей позиции
	В Номенклатура работы специалиста выбрать номенклатуру, которая будет назначена стандартной для соответствующей позиции
7.	В справочнике Договоры контрагентов ввести договоры на абонентское обслуживание, в которых необходимо в качестве вида договора выбрать Абонентское обслуживание и заполнить его условия
8.	Произвести базовые настройки телеграм-бота для возможности отправки сообщений специалистам об изменениях в их графиках работ:
	В Токен управления телеграм ботом ввести токен управления телеграм ботом, созданного для оповещения сотрудников
	В Идентификатор группы для оповещения ввести индентификатор группы нашей компании
	Добавить сотрудников, которым должны приходить оповещения, в данную группу
